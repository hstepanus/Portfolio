NOVEC Predictive Grid

An interactive clean-energy planning dashboard built to visualize EV adoption, solar density, infrastructure patterns, hotspots, and spatial clusters across the NOVEC service area. The project combines geospatial analytics with an executive-facing web experience.

Web GIS • Energy Infrastructure • Spatial Statistics

Decision support for distributed energy planning

Built for planners, engineers, analysts, and leadership, this dashboard turns raw EV, solar, and charging data into map-based intelligence. It highlights where adoption is clustering, where infrastructure gaps may exist, and how spatial patterns can support proactive utility planning.

PostgreSQL / PostGIS Leaflet Hotspot Analysis LISA Clusters Responsive Dashboard
Launch Live App View GitHub Repository View Resume

Problem Solved

Utilities often have totals, spreadsheets, and disconnected reports, but not a clear spatial view of where clean-energy growth is concentrated. This project makes those patterns visible, measurable, and actionable.

Key Features

  • Layered EV, solar, service area, and charger datasets
  • Executive-style dashboard layout with responsive design
  • Hotspot and cluster outputs for planning insight
  • Visual storytelling for non-technical stakeholders

Stack

ArcGIS Pro, PostGIS, Python, Leaflet, GeoJSON, GitHub Pages, and cartographic design principles shaped the final product and presentation-ready UI.

Impact

The result is a portfolio-level GIS product that communicates technical capability and executive clarity at the same time, making it suitable for hiring managers, mentors, and utility audiences.
